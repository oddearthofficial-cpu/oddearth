Privacy Policy

Last updated: April 1, 2026

1. Introduction

This Privacy Policy describes how OddEarth Pipeline ("we", "our", or "the app") handles information when you use our content automation tool. OddEarth Pipeline is a private automation system used exclusively to manage and publish content to the OddEarth social media channels on YouTube, TikTok, and Instagram.

2. Information We Access

To perform automated uploads, OddEarth Pipeline accesses the following:

  • YouTube: OAuth 2.0 credentials (access token and refresh token) to upload videos, set thumbnails, and post comments on the OddEarth YouTube channel.
  • TikTok: OAuth 2.0 access credentials to publish short-form video content to the OddEarth TikTok account.
  • Instagram (Meta): OAuth 2.0 access credentials via the Instagram Graph API to publish Reels to the OddEarth Instagram account.

These credentials are stored locally on the server operating the pipeline and are never transmitted to any third-party service, database, or analytics platform.

3. How We Use This Information

The credentials and account access listed above are used solely for the following purposes:

  • Automatically uploading generated video content to the designated social media accounts.
  • Setting video metadata (titles, descriptions, tags, thumbnails).
  • Posting pinned comments on uploaded videos.
  • Monitoring upload status and quota usage.

We do not use this access for advertising, analytics, data brokering, or any purpose beyond direct content publishing on behalf of the OddEarth channel owner.

4. Data Storage and Security

Authentication tokens are stored locally in configuration files on the private server running OddEarth Pipeline. These files are not exposed to the public internet. We use environment variables and restricted file permissions to protect sensitive credentials.

No user data, viewer information, or analytics data is collected, stored, or processed by OddEarth Pipeline.

5. Data Sharing

We do not sell, trade, rent, or otherwise share any credentials, account data, or personal information with any third parties. All API interactions occur directly between our private server and the respective platform APIs (Google/YouTube, TikTok, Meta/Instagram).

6. Third-Party Services

OddEarth Pipeline interacts with the following third-party APIs:

  • Google / YouTube Data API v3 — for video uploads and channel management.
  • TikTok Content Posting API — for publishing short-form video content.
  • Meta Instagram Graph API — for publishing Instagram Reels.
  • ElevenLabs API — for AI voice generation (no personal data is sent).
  • Pexels API — for royalty-free stock footage downloads (no personal data is sent).
  • Anthropic API — for AI script generation (no personal data is sent).

Each of these services is governed by their own privacy policies. OddEarth Pipeline only transmits content assets and metadata required for publishing.

7. Data Retention

Authentication tokens are retained locally as long as the OddEarth Pipeline system is operational. Tokens can be revoked at any time by the channel owner via the respective platform's account settings. Upon revocation, the pipeline ceases to function until re-authenticated.

8. Children's Privacy

OddEarth Pipeline is a private tool operated by the channel owner and is not directed at or accessible to children. We do not knowingly collect any information from minors.

9. Changes to This Policy

We may update this Privacy Policy periodically. Any changes will be reflected with an updated "Last updated" date at the top of this page.
