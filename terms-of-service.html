2. Description of Service

OddEarth Pipeline is a private content automation tool used exclusively by the OddEarth channel owner to:

  • Generate AI-assisted video scripts using the Anthropic Claude API.
  • Synthesize narration audio using the ElevenLabs text-to-speech API.
  • Download royalty-free stock footage from Pexels.
  • Assemble final video files using FFmpeg and MoviePy.
  • Automatically upload finished videos to YouTube, TikTok, and Instagram on a scheduled basis.

OddEarth Pipeline is not a public-facing consumer product. It is a personal automation system operated solely for managing the OddEarth social media presence.
